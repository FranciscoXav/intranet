

Decidi desenvolver um projeto, ainda numa fase muito embrionária, que me permita, no futuro, criar uma aplicação para utilização em contexto real.

O trabalho consiste numa plataforma destinada a registar a autorregulação dos alunos dos cursos profissionais. Esta plataforma inclui uma página inicial que autoriza os utilizadores através das suas credenciais. No momento, a aplicação permite apenas que os alunos registem informação, sendo permitidas apenas as seguintes credenciais:

	student1.esct@gmail.com
	Aluno01%

	student2.esct@gmail.com
	Aluno02%

Após o login na plataforma, o aluno, previamente inscrito num curso e turma, poderá selecionar a disciplina, o módulo e o respetivo conteúdo para o qual pretende efetuar a sua autorregulação.

Relativamente aos ficheiros JSON armazenados na pasta "src/dados/json/", apenas um (registos.json) é acedido através de uma API REST, cujo servidor deve ser iniciado na porta 4000 e com o seguinte endpoint http://localhost:4000/registos.
